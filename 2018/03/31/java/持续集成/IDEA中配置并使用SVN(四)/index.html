idea中配置并使用SVN(四)

发表于 2018-03-31 | 分类于 持续集成

持续集成之本地安装 tortoisesvn 客户端
配置 idea 与 svn 集成与使用…

1、下载 tortoisesvn 客户端 https://tortoisesvn.net/downloads.html


七牛云：TortoiseSVN-1.9.7.27907-x64-svn-1.9.7.msi

2、安装 tortoisesvn 客户端

  • command inne client tools 默认不安装，需要安装上

3、idea 中配置 svn 路径

4、启用版本控制集成，选择 subversion

5、分享 svn 目录

6、测试上传文件到svn VCS–>Subversion–>Commit Directory

7、idea 中设置 svn 忽略文件或文件夹（不上传到 svn），比如忽略项目中的 target 目录

  • 进入version control 控制台
  • 点击 Configure Ignored Files 图标
  • 点 + 号添加文件或文件夹

  • ignore specified file：忽略指定文件
  • ignore all files under：忽略选项下的所有文件
  • ignore all files matching：忽略所有匹配文件

7、提交代码测试 target 目录是否成功被忽略，无 target 目录显示出来则证明忽略成功
