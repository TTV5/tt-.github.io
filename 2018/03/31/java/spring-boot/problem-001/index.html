spring boot 读取 application.properties 配置文件中文乱码

发表于 2018-03-31 | 分类于 spring boot

spring boot 读取 application.properties 配置文件中文乱码…

问题描述

在集成spring boot web的时候,在 application.properties 配置文件中定义了中文的值,在代码中获取这个值的时候出现了乱码

解决方案

统一字符集编码为 utf-8
在 idea 中查看 application.properties 配置文件的编码是否为 utf-8

点击 file-setting

点击 Editor-File-Encodings ,我这里的配置文件是 gbk
设置默认字符集为 utf-8
勾选 Transparent native-to-ascii conversion
然后 ok 保存
